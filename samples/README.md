# Data Samples

This folder contains small extracts from the public datasets used in this
portfolio project. Full raw datasets are kept out of GitHub; see
`docs/data_sources.md` for source and download information.

These samples demonstrate data structure only. They do not contain ExxonMobil,
PNG LNG, confidential, proprietary, or licensee data.

## Azure Predictive Maintenance Source Samples

The source-structure samples in `samples/azure/` are generated by:

```cmd
python scripts\inspect_azure_pdm.py
```

Each sample is capped at 100 rows.

| Sample File | Source File | Rows | Description |
|---|---|---:|---|
| `azure/PdM_machines_sample.csv` | `PdM_machines.csv` | 100 | Machine catalog with machine ID, model, and age |
| `azure/PdM_telemetry_sample.csv` | `PdM_telemetry.csv` | 100 | Hourly sensor readings for voltage, rotation, pressure, and vibration |
| `azure/PdM_errors_sample.csv` | `PdM_errors.csv` | 100 | Error event records by timestamp, machine, and error ID |
| `azure/PdM_failures_sample.csv` | `PdM_failures.csv` | 100 | Failure event records by timestamp, machine, and failed component |
| `azure/PdM_maint_sample.csv` | `PdM_maint.csv` | 100 | Maintenance event records by timestamp, machine, and component |

## Azure Reliability Summary Sample

The dashboard-ready reliability summary sample is generated by:

```cmd
python scripts\build_azure_reliability_summary.py
```

| Sample File | Source | Rows | Description |
|---|---|---:|---|
| `azure/azure_machine_reliability_summary_sample.csv` | Azure PdM processed summary | 100 | One row per machine with LNG-style asset mapping, reliability-style KPIs, health score, priority, and engineering note |
