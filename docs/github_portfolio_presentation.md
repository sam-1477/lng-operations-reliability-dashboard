# GitHub Portfolio Presentation

## Project Title

LNG Operations Technical Reliability Dashboard

## Project Summary

This project is an operations technical reliability dashboard built with Python
and Excel using public industrial datasets adapted into LNG-style reliability
categories. It demonstrates asset reliability screening, compressor process
surveillance, PHMSA incident risk awareness, and engineering communication for a
graduate mechanical engineering portfolio. It is a portfolio demonstration, not
ExxonMobil or PNG LNG operating data.

## Who The Project Is For

This project is aimed at reviewers interested in graduate mechanical
engineering, LNG Operations Technical support, maintenance engineering,
reliability engineering, rotating equipment, mechanical integrity, and
data-driven engineering communication.

It is especially relevant for:

- Graduate Mechanical Engineer applications
- LNG Operations Technical or reliability-focused roles
- Maintenance and asset integrity portfolio discussion
- GitHub, CV, and LinkedIn technical project presentation

## Relevance To LNG Operations Technical Support

LNG Operations Technical support involves process surveillance, equipment
reliability, maintenance prioritization, incident awareness, mechanical
integrity thinking, and clear communication with operations and maintenance
teams. This project uses public datasets to demonstrate those skills without
using confidential plant information.

The dashboard connects:

- Asset reliability and maintenance priority from Azure Predictive Maintenance
- Compressor monitoring concepts from MetroPT-3
- LNG and pipeline incident risk awareness from PHMSA
- Excel-based reporting for non-Python reviewers

## Public Datasets Used

| Dataset | Source | Dashboard Use |
|---|---|---|
| Microsoft Azure Predictive Maintenance Dataset | Microsoft / Kaggle public dataset | Asset health, maintenance priority, reliability-style KPIs |
| MetroPT-3 Compressor Dataset | Public compressor dataset from UCI / associated publication | Compressor process surveillance and rotating equipment monitoring |
| PHMSA LNG and Gas Transmission Incident Data | U.S. DOT PHMSA public incident data | LNG incident awareness, pipeline incident trends, cause review, consequence context |

## Honest Limitations

- This project uses public industrial datasets adapted into LNG-style
  reliability categories.
- It is not ExxonMobil or PNG LNG operating data.
- It is not a live plant dashboard.
- It is not a formal mechanical integrity assessment.
- The Azure MTBF-style, MTTR-style, and availability-style values are estimates
  for portfolio learning.
- MetroPT-3 is an air compressor dataset, not an LNG refrigeration compressor.
- PHMSA data is U.S. public incident data, not PNG LNG operating history.
- The Excel workbook is currently a dashboard skeleton, not a final polished
  production dashboard.
