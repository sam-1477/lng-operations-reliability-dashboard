# Project Charter
## LNG Operations Technical Reliability Dashboard

---

## Project Vision

A GitHub portfolio project that demonstrates operations technical support and reliability
engineering thinking — the exact skills ExxonMobil PNG seeks in a Graduate Engineer for
their Operations Technical team.

## Problem Statement

A fresh graduate applying for a technical operations role in the LNG sector faces a
challenge: how do you demonstrate reliability engineering capability without access to
real LNG plant data?

This project solves that by adapting **publicly available industrial datasets** into an
LNG-style reliability dashboard. It proves the candidate can think in terms of asset
hierarchies, reliability KPIs, condition monitoring, pipeline risk, and maintenance
prioritization — without ever touching confidential data.

## Target Audience

1. **Primary:** ExxonMobil PNG hiring panel reviewing graduate engineer applications
2. **Secondary:** Other LNG/energy sector recruiters
3. **Tertiary:** Fellow engineering students and graduates building portfolio projects

## Success Criteria

The project succeeds if a reviewer can answer "yes" to all of these:

- [ ] Does the candidate understand reliability engineering fundamentals?
- [ ] Can the candidate structure and analyze industrial data?
- [ ] Does the candidate present technical work professionally?
- [ ] Is the candidate honest about data sources and limitations?
- [ ] Would I be confident assigning this person to support operations technical work?

## Guiding Principles

1. **Honesty First:** Every chart, every number, every recommendation must be traceable
   to a public data source or clearly labeled as adapted/estimated. Never blur the line
   between adapted data and real LNG operations data.
2. **Beginner-Friendly:** The project must be buildable by a fresh graduate on a Windows
   laptop. No Docker, no cloud infrastructure, no complex deployment.
3. **Reviewer-Ready:** The Excel workbook must stand alone — a non-Python reviewer can
   open it and understand the results.
4. **Industry-Relevant:** Equipment categories, KPIs, and analysis methods must reflect
   what is actually used in LNG operations technical support.

## Stakeholders

| Stakeholder | Interest |
|-------------|----------|
| Candidate (You) | Portfolio asset for job applications |
| ExxonMobil PNG Hiring Panel | Evidence of engineering capability |
| GitHub Community | Open-source reliability dashboard template |
| Future Employers | Demonstration of data analysis and engineering skills |

## Constraints

- No ExxonMobil or PNG LNG confidential data
- No real-time data feeds
- Windows 11 laptop with 8+ GB RAM
- No budget for paid APIs (use free tiers)
- Must complete within 2 weeks (part-time)

## Deliverables

1. Public GitHub repository with complete source code and documentation
2. Excel workbook containing all data, charts, and KPIs
3. Jupyter notebooks demonstrating step-by-step analysis
4. Screenshots suitable for CV and LinkedIn
5. Architecture plan (this document set)
