# Screenshot Guide

This folder contains dashboard screenshots for quick recruiter and engineering
reviewer inspection. The screenshots are provided so a reviewer can understand
the Excel dashboard skeleton without downloading or opening the workbook first.

The project uses public industrial datasets adapted into LNG-style reliability
categories for portfolio demonstration. The dashboard is not ExxonMobil or PNG
LNG operating data, not a live plant dashboard, and not a formal mechanical
integrity assessment.

## Files

| Screenshot | Description |
|---|---|
| `dashboard_preview.png` | Main dashboard preview with KPI cards and starter charts for the operations technical reliability dashboard |
| `kpi_calculations.png` | KPI calculation sheet showing summary values and helper tables used by the dashboard |
| `ai_recommendations.png` | AI recommendations starter table with cautious engineering review language |

## Reviewer Use

Start with `dashboard_preview.png` for the overall visual impression. Then use
`kpi_calculations.png` to understand the dashboard metrics and
`ai_recommendations.png` to review the recommendation framing and caveats.

For the full workbook, open:

`excel/lng_operations_reliability_dashboard.xlsx`
